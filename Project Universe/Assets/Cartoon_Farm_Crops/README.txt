-HOW TO SET UP THIS PACKAGE-

Step 1: Import the "Effects" package from the standard assets into your project.

Step 2: Select all materials in the "Materials" folder of the "Cartoon_Farm_Crops" package and change their shader to "Toon/Lit Outline".

Step 3: Set the "Outline width" sliver to "0.002".

Step 4: Change the "Toon Ramp (RGB)" to "UtilToonGradient.png"
